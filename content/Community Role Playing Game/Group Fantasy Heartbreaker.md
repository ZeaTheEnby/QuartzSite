# Overview
The goal of this project is to create a custom DND style fantasy adventure game together.

Goals of System
* Customizable - should be easy to create different kinds of characters.
* Modular - should be able to  take parts out with out the whole thing breaking
* Compatible - should be somewhat compatible with different styles and systems

Basic system (generic DND style Rolls)
* d20 system with the 6 DND attributes.
* Roll above target number while adding your skill to the role.
* Featuring advantage and disadvantage.
* Only roll if both results could be interesting

# Goals
The goal of this idea is to make our own hack of a DND style game by just adding a variety of rules to it about anything and everything people find interesting.
There are some key concepts for this game specefically that need to be established so everyone is on the same page.

### Key Concepts
* Low power level characters - Characters will have unique abilities but few will be stat increases, players are always meant to be squishy though competent but not god tier heroes.
